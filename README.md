# Transformers

Transformer attention mechanisms implemented from scratch in PyTorch.

## Contents

| File | Description |
|------|-------------|
| `attention.py` | Scaled dot-product attention |
| `multihead_attention.py` | Multi-head attention |
| `masks.py` | Padding mask and causal (look-ahead) mask |
| `positional_encoding.py` | Fixed sinusoidal positional encoding |
| `why_sqrt_dk.py` | Intuition behind the `1/√d_k` scaling factor |

## Requirements

```bash
pip install torch
```
